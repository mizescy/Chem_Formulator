## About The Project

**Chemical Formulator** is a cutting-edge desktop application designed for chemical component analysis and essential oil identification. Built with modern technologies and enhanced by machine learning algorithms, this tool provides accurate matching of chemical components against a comprehensive database of marker components.


### üèóÔ∏è Architecture
The application features a robust, object-oriented architecture that ensures:
- **Scalability** - Easy to extend with new features and databases
- **Maintainability** - Clean, well-documented code structure
- **Performance** - Optimized algorithms for fast analysis
- **User Experience** - Intuitive interface with comprehensive feedback

## Getting Started

1. **üìä Setup Database**
   - Obtain your database in `.xlsx` format
   - Drag and drop the file into the correct directory
   
   ![Database Setup][product-directory]

2. **üîß Generate Marker Files**
   - Run the marker file generator for the formulator to function properly
   - This step is crucial for component recognition
   
   ![Marker Generation][product-marker]

> **‚ö†Ô∏è Important Note**: Whenever new data is added to the database, you must redownload your database and redo step 1-2 over again to ensure the application recognizes the latest data.

<!-- USAGE GUIDE -->
## Usage Guide

### Step-by-Step Analysis Process
> **‚ö†Ô∏è Important Note**: Performance varies with different System Specs some will be slow some will be fast to get the analysis.

#### 1. **Input Chemical Data**
- Copy your raw materials and their percentages
- Paste the data into the text area
- Click the "Analyze" button to begin processing

![Analysis Process][product-analysis]

#### 2. **Select Components**
- Review the analysis results
- Choose your preferred bases, essential oils, and other components
- Confirm your selections

#### 3. **Export Results**
- Click the export formula icon
- Select your desired save location
- Generate professional reports with detailed annotations

![Export Process][product-end]

### üí° Pro Tips
- **Batch Processing**: Process multiple formulations simultaneously for efficiency
- **Data Validation**: Always verify input data format before analysis
- **Regular Updates**: Keep your database current for optimal accuracy

<!-- SYSTEM REQUIREMENTS -->
## System Requirements

### Minimum Requirements
- **OS**: Windows 10, macOS 10.14, or Linux Ubuntu 18.04+
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 2GB available space
- **Processor**: Intel i3 or AMD equivalent

### Recommended Specifications
- **OS**: Latest stable versions
- **RAM**: 16GB for large dataset processing
- **Storage**: SSD with 5GB+ available space
- **Processor**: Intel i5/i7 or AMD Ryzen 5/7
