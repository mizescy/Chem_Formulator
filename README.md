## About The Project

Chemical Formulator is an advanced chemical component analysis application that enables users to identify potential essential oil matches by analyzing input chemical components against a comprehensive database of marker components. The tool features a modern, object-oriented architecture with enhanced code readability and maintainability for future development.

### Key Capabilities:
* **Chemical Component Analysis** - Input chemical components and receive potential essential oil matches
* **Machine Learning Integration** - Enhanced identification accuracy through trained ML models
* **Excel Integration** - Import data from Excel files and export results with detailed annotations
* **Advanced Matching System** - Display detailed match scores based on the number and accuracy of marker components matched

## Usage Guide
1. Copy both Raw materials and its percentage and paste it in the text area and press analyze.
![product-analysis]

2. Choose which bases, essential oils etc of your choice, confirm you choices and press export formula icon, will prompt you to select you save location. 
![product-end]

### Extra information
- Everytime if there are new data added to the database will need to predefine the markers in bold and re-run make marker again for it to register the latest data and for the app to know
- machine prediction is only data up to as of June 2025
